# Videos de Meteoro - Einherjar Blitz

Esta carpeta contiene los videos de meteoro para diferentes rarezas.

## Archivos necesarios:
- meteor-3star.mp4 - Video para recompensas comunes (1-3 estrellas)
- meteor-4star.mp4 - Video para recompensas raras/épicas (4 estrellas)
- meteor-5star.mp4 - Video para recompensas legendarias/míticas (5 estrellas)

## Nota:
Actualmente el sistema usa simulaciones CSS en lugar de videos reales.
Los videos pueden ser agregados posteriormente reemplazando las simulaciones
en el archivo genshin-system.js.

## Especificaciones de video recomendadas:
- Formato: MP4 (H.264)
- Resolución: 1920x1080 o superior
- Duración: 2-4 segundos
- Formato de audio: AAC (opcional)
- Bitrate: 5-10 Mbps
